高一的时候还没有（至少我还接触不到）那些听译软件，我一字一句地把一个英文教程啃了一大半发了出来（虽然后面弃坑了），现在突然想把以前存的那些教程翻译一下，然后我试了试一个开源的工具，结果效果挺不理想的，首先是pyinstaller打包特有的加载一堆依赖使得软件非常臃肿；还有是操作逻辑，如果对源码不太了解的话有些功能还是会挺令人迷惑的（比如通过OpenAI调用API之类的）。鉴于现在各种听译模型非常成熟，我的想法是重新开发一个更加简化的听译软件，争取在这个寒假搞出来吧 [Mygo表情包_害羞] 也许之后会发个github repo



## 1. 项目愿景 (Project Vision)

打造一款开源、轻量级、模块化的跨平台视频字幕生成与翻译工具。旨在解决传统 Python AI 应用体积臃肿、部署困难的问题，同时兼顾本地高性能（CUDA）与云端 API 的灵活性。

软件功能：用户提交一个视频，选择视频语言与需要翻译语言，并生成双语字幕。生成的字幕可以编辑。初期暂不提供渲染字幕/自定义的功能。

视觉层面：一个GUI，入口需要尽量简洁，仅提供必要的添加视频，生成字幕等按钮，高级选项整合到高级菜单中

软件规模：本体尽量小巧，提供额外添加模型的功能

技术栈：语言期望使用Go作为主要语言，可选CUDA作为硬件加速功能。

## 4. GUI

目录位于`gui/`，包括主界面定义 `main_window.go` ，其他模块、按钮等均通过接口注册在main_window中。

软件提供多语言功能(包括中文、英文)，通过加载`config/` 下的`lang-zh.json`, `lang-en.json` 实现界面语言更改。



## 5. 接口定义
